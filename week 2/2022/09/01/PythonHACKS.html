"Residence": "San Diego",

Iterating through Dictionary using "For Loop" 

for key in InfoDB:
        print(str(key) + " : " + str(InfoDB[key]))

Iterating through Dictionary using "While Loop" 

keynumber = 0
valuenumber = 0
while keynumber < len(InfoDB) :
    key = list(InfoDB)[keynumber]
    val = list(InfoDB.values())[valuenumber]

    keynumber += 1
    valuenumber += 1
